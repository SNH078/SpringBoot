# hibernate   
ORM (Object Relational Mapping) framework for Java. 

# Why Hibernate?  
note - java is oops -> returns objects  , JDBC requires sql to keep the object data in db
- Now, to avoid writing SQL
- to map Java objects to database tables automatically

# 🔄 What Hibernate does  
Java Object ↔ Hibernate ↔ Database Table  
👉 This is called ORM (Object Relational Mapping)  

🧩 In Spring Boot  
👉 You don’t use Hibernate directly  
You use:  
Spring Data JPA

👉 Internally:  
Spring Data JPA → Hibernate → JDBC → Database
