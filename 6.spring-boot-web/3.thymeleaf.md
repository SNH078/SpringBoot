## converting our jsp project in thymeleaf

- remove application.property (don't need in this )   
- .jsp files-> .html  and move them to "templates" folder unsder resources folder
- add dependency for thymeleaf
  `	<dependency>
			<groupId>org.springframework.boot</groupId>
			<artifactId>spring-boot-starter-thymeleaf</artifactId>
		</dependency>`

  -  <img width="361" height="282" alt="image" src="https://github.com/user-attachments/assets/9fe704c2-9f96-4681-a372-13148da461c6" />
remove this line from .html files  and add this
`<html xmlns:th="http://www.thymeleaf.org">`

- use thymeleaf syntax for printing value of variable in html files
