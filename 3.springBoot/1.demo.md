### created a Spring-managed object (Alien) and retrieved it from the container instead of creating it manually & used wiring... demonstrating IoC & DI 

#### wiring   
Wiring = Connecting objects (beans) together  

If one class needs another class, Spring will:  
Create both objects  
Connect them automatically  
👉 That connection = Wiring
