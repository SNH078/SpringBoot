# JDBC (Java Database Connectivity)
API that allows Java to connect to and interact with a database.  

## What it does  
Connects Java → Database  
Executes SQL queries  
Retrieves results  

# Steps   
```
Load driver → connect → prepare → execute → process → close
```

```java
import java.sql.*;

public class Main {
    public static void main(String[] args) {

        String url = "jdbc:postgresql://localhost:5432/your_db";
        String user = "postgres";
        String password = "your_password";

        try {
            // 1. Create Connection
            Connection con = DriverManager.getConnection(url, user, password);
            System.out.println("Connected to DB!");

            // 2. Create Table (optional)
            Statement st = con.createStatement();
            st.executeUpdate("CREATE TABLE IF NOT EXISTS student(id SERIAL PRIMARY KEY, name VARCHAR(50))");

            // 3. Insert Data
            String insertQuery = "INSERT INTO student(name) VALUES (?)";
            PreparedStatement ps = con.prepareStatement(insertQuery);
            ps.setString(1, "John");
            ps.executeUpdate();

            // 4. Fetch Data
            ResultSet rs = st.executeQuery("SELECT * FROM student");

            while (rs.next()) {
                int id = rs.getInt("id");
                String name = rs.getString("name");
                System.out.println(id + " - " + name);
            }

            // 5. Close Connection
            con.close();

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```
